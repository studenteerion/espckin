Installazione

Requisiti

  • Node.js (versione 14 o successiva)
  • npm (Node Package Manager)

Istruzioni

Per installare l'applicazione, segui questi passaggi:

  1. Clona il repository: git clone https://github.com/tuo-repo.git
  2. Installa le dipendenze: npm install
  3. Avvia il server: npm start
  4. Visita http://127.0.0.1:9898 nel tuo browser

Endpoint

  • GET /api/v1/resources - Recupera tutte le risorse

  • GET /api/v1/resources/:id - Recupera una risorsa specifica
